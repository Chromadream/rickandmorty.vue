# Simple Rick and Morty

Tried to learn Vue.js, here is the result.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
